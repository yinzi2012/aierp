# AIERP · 智能制造业 ERP 系统功能介绍

> **专为中小制造企业打造 · 业务全链可追溯 · AI 驱动智能决策**

---

## 一、系统定位

很多制造企业正在被同一批问题困扰：

- 订单来了，原材料够不够？谁也说不清楚
- 生产到哪一步了？只能靠车间主任打电话
- 出了质量问题，用的是哪一批料？无从追查
- 应收账款多少、账期多长？财务只能翻 Excel
- 想看业务数据，要等 IT 出报表，往往已经晚了

**AIERP** 就是为解决这些痛点而生的。它以**全链路数字追溯**为核心，融合**AI 自然语言查询**，将销售、生产、采购、仓储、财务打通为一个有机整体，让每一张单据都有迹可循，让每一个决策都有数据支撑。

---

## 二、核心痛点解决方案

| 客户痛点 | AIERP 解决方式 |
|--------|--------------|
| 订单与生产脱节，交期难保障 | 销售订单一键触发生产计划，自动 BOM 展开计算物料需求 |
| 生产过程不透明，进度靠打电话 | 工单精确到工序，工人扫码报工，实时进度一屏掌握 |
| 采购计划拍脑袋，经常缺料或积压 | 基于 BOM 和生产计划自动测算需求缺口，生成精准采购建议 |
| 原材料来源不清，质量问题无法溯源 | 每批物料从入库起携带溯源码，质检状态全程留痕 |
| 财务与业务数据两张皮 | 入库自动挂应付、出库自动挂应收，业务动作即财务凭证 |
| 数据分析靠 Excel，周期长还出错 | AI 自然语言查询，用白话问系统，秒级返回分析结果 |

## 三、功能模块详解

### 1. 销售管理

**从签单到回款，全程线上闭环**

- 销售合同 → 销售订单，状态实时流转（草稿 / 已审核 / 执行中 / 完结）
- 订单审核后**自动锁定**，防止误修改；支持按客户、产品、时间多维度检索
- 与生产计划深度联动：销售订单一键下达，系统自动判断是否已有在产计划，**防止重复生产**
- 销售数量与出库数量实时回写，交货进度一目了然
- 全链路追溯：从销售订单可一键查看 → 生产计划 → 工单 → 领料 → 完工 → 出库 → 收款的完整链条

---

### 2. 生产管理（计划 + 订单 + 工单）

**三层管控，让生产过程透明可控**

**生产计划层**
- 对接销售订单，自动关联目标 BOM，一键展开物料清单
- 齐套检查：自动对比现有库存与需求，给出缺料清单
- 缺料自动生成采购计划，无需手工填写
- 支持多销售订单合并生产，节省换产成本

**生产订单层**
- 从生产计划下达生产订单，自动带入产品信息、物料明细、工序明细
- 工序明细支持自定义排序，关联工时定额和工序成本

**生产工单层（精细化车间管理）**
- 按工序批量生成工单，自动分配车间、生产线、班组
- 支持甘特图排程，可视化管理每条产线负荷
- 工人登录即识别身份，只看本班组工单，操作简洁
- 多次报工累计，完工率自动计算，达标后工单自动关闭
- 工单状态机：待下发 → 已下发 → 进行中 → 已完成

---

### 3. 采购管理

**需求驱动，全程可控，来源可查**

- 采购计划从生产需求自动生成，数量精准不依赖经验
- 采购订单支持多供应商询价对比，锁价后审核
- 到货管理：收货数量、质量状态（合格 / 不合格 / 让步接收）逐批记录
- 支持**采购退货**全流程：拒收 → 退货通知 → 供应商取货，状态自动流转
- 采购完成后库存自动更新，无需手工录入

---

### 4. BOM 物料清单管理

**多层嵌套，精准算料，成本透明**

- 支持无限层级树形 BOM 结构，适应复杂装配产品
- BOM 版本管理，变更历史完整保留
- 一键 BOM 展开：自动计算各层级物料需求数量
- BOM 价格计算：自动汇总原材料成本，辅助定价决策
- BOM 变更实时同步至生产订单和采购计划，避免用错版本

---

### 5. 仓储管理（入库 / 出库 / 库存）

**库存账实相符，出入有迹可查**

- 采购入库、生产完工入库、销售退货入库，多类型入库统一管理
- 生产领料出库、销售发货出库，每笔出库都关联来源单据
- 库存流水永久留痕（`stock_log`），支持按物料、时间、仓库查询历史
- 库存预警：低于安全库存自动提醒，关键物料不断货
- 出库时自动回写应收单，实现业财同步

---

### 6. 财务管理（应收 / 应付）

**业务驱动财务，告别手工台账**

**应付账款**
- 采购入库单 → 自动生成应付单，金额、数量、供应商信息自动带入
- 支持一张采购单多次付款
- 账龄分析：按 30/60/90/180 天区间统计欠款，逾期自动预警

**应收账款**
- 销售出库单 → 自动生成应收单，与销售订单全程关联
- 收款单核销：支持部分收款、多次收款，核销状态实时更新
- 一张应收单可追溯回源头销售合同、出库记录、客户信息

---

### 7. 全链路追溯系统

**这是 AIERP 的核心差异化能力**

制造业最怕两件事：**质量问题不知道根源**，**成本核算算不清楚**。追溯系统直接解决这两个问题。

**追溯路径（10+ 节点完整链条）：**

```
销售订单 → 生产计划 → 采购计划 → 采购订单 → 采购到货
    → 采购入库 → 生产领料 → 生产工单 → 完工入库
    → 销售出库 → 应收单 → 收款单
```

**4 种追溯入口：**
- 从**销售订单**追：这批货用了哪些料？经过了哪些工序？
- 从**采购订单**追：这批物料用在了哪些产品上？
- 从**物料编码**追：某型号零件的所有批次来源与去向
- 从**财务单据**追：这笔应收/应付对应的业务全貌

**可视化展现：**
- SVG 流程图：节点状态用颜色区分，支持缩放和导出
- 垂直时间轴：各节点发生时间顺序一览
- 成本分析饼图：物料成本 / 人工成本 / 制造费用直观呈现

---

### 8. AI 智能助手

**让业务人员直接和数据对话**

- **自然语言查询（NLQ）**：用白话提问，系统自动转换为 SQL 查询并返回结果
  - "上个月哪个产品最赚钱？"
  - "本周入库最多的前 5 种物料是什么？"
  - "当前有多少张采购订单还没有到货？"
  - "哪个供应商的采购金额最高？"
- 支持结果以表格展示，支持 SQL 调试模式（管理员可见）
- AI 服务独立部署（Python FastAPI + LangChain），不占用主系统资源
- 复用系统统一 JWT 认证，无需重新登录

## 五、适用场景

AIERP 特别适合以下类型的制造企业：

- **离散制造**：机械设备、精密零部件、电子组装
- **多工序生产**：需要工序排程和工单管理
- **供应链复杂**：多供应商、多层次 BOM 产品
- **质量要求高**：需要物料批次追溯和质检管理
- **规模**：50 ~ 500 人，已有一定信息化基础但需要升级
